General Page Set Up

CSS Element Result
margin: Adds left and right margin to specified area. This can also be set to "auto" to center elements.
background-color: Set background color of full page or section/element.
padding: Add space around an item. This can be specified as a general value, or specific for each side of an element.

Font Options

CSS Element Result
font-family: Sets font for full document, sections, or individual types of text. Available fonts can be found online.
font-size: Sets font size based on pixedl height. Typically used to set text size for individual types of text.
font-weight: Determines boldness of text.
text-align: Sets the horizontal text justification. Typically used to set justification for full sections of text.
